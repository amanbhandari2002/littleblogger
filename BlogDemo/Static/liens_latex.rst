Astuces
=======

- Insérer une image et mettre du texte a coté (à vous de changer les 
  proportions données ici comme 50/50):

    .. sourcecode:: latex

        \begin{minipage}{0.5\linewidth}
        Mon image ici
        \end{minipage}\hfill
        \begin{minipage}{0.5\linewidth}
        Mon texte là
        \end{minipage}
